# CareConnect AI

Healthcare management system with conversational AI, RAG, and multi-portal architecture.

## Project Details
- **Duration**: 14 days
- **Technologies**: Python, Streamlit, FastAPI, Ollama, ChromaDB, SQLite
- **Cost**: Zero (local development)

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Start Ollama: `ollama serve`
3. Run frontend: `streamlit run frontend/main_app.py`

## Project Structure
- `backend/` - FastAPI server, database, LLM integration
- `frontend/` - Streamlit portals for different users
- `data/` - SQLite DB, vector store, uploads
- `notebooks/` - Exploration and testing
